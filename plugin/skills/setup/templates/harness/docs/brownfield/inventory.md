## Format

For each area:

- **Purpose**: what it does (verified / inferred)
- **Dependencies**: what it depends on, what depends on it
- **Tests**: existing test coverage (good / partial / none)
- **Risk**: known risks or concerns
- **Status**: mapped / partially mapped / unmapped
