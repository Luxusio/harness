# Purpose
This file defines architecture boundaries verified by check-architecture.sh.
Every rule here MUST have a corresponding check in the script. A rule without enforcement is just a suggestion.

# How to use
1. Define rules below (uncomment examples or write your own)
2. Add corresponding grep/check in check-architecture.sh
3. critic-runtime and maintain will run the checker automatically
4. Violations are FAIL conditions, not warnings
