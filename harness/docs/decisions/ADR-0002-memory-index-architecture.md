# ADR-0002: Shared compiled memory index architecture

**Status:** accepted
**Date:** 2026-03-24

## Context
The harness repo-local operating system needs a way to share compiled memory across sessions.

## Decision
Introduce a deterministic compiled memory index under harness/memory-index/ that is committed to git.

## Consequences
- All users share the same retrieval advantage
- Source of truth remains in harness/docs/*, harness/state/*, harness/policies/*
- Index is regenerated by script, never manually edited
